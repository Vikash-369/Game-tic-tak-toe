        <h1>Overview</h1>
        <p>A simple and interactive Tic Tac Toe game built using HTML, CSS, and JavaScript. The game features smooth gameplay, turn indication,
             and a clean UI with a modern gradient background. It’s
              a fun way to practice JavaScript logic, event handling, and DOM manipulation.</p>
              <video src="[Tic-Tac-Toe - Google Chrome 2025-11-12 18-52-45.mp4](https://github.com/Vikash-369/Game-tic-tak-toe/blob/main/Tic-Tac-Toy.mp4)" autoplay muted></video>
             <h2>Features</h2>
             <ul>
                <li><b>Two-Player Mode:</b> Play between “X” and “O”.</li>
                <li><b>Turn Indicator:</b> Displays whose turn it is in real-time.</li>
                <li><b>Winner Detection</b>: Automatically checks and announces the winner or draw.</li>
                <li><b>Restart Option:</b> Quickly restart the game with a single click.</li>
                <li><b>Responsive Design:</b> Works smoothly on both desktop and mobile screens.</li>
                <li><b>Gradient UI:</b> Soft and minimal gradient background for a polished look.</li>
             </ul>
             <h2>Tech Stack</h2>
             <ul>
                <li><b>HTML5 –</b> Game structure</li>
                <li><b>CSS3 –</b>Styling and layout</li>
                <li><b>JavaScript (ES6) –</b>Game logic and interactivity</li>
             </ul>
             <h2>How to Play</h2>
             <ul>
                <li>Open the project folder in your browser (index.html).</li>
                <li>Players take turns clicking the boxes.</li>
                <li>The game announces the winner or draw automatically.</li>
                <li>Click “Restart Game” to play again.</li>
             </ul>
